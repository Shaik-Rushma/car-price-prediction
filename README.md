# Car Price Prediction using Machine Learning

## Overview

This project predicts used car prices using Machine Learning based on features such as:

* Present Price
* Kilometers Driven
* Fuel Type
* Seller Type
* Transmission
* Number of Owners
* Car Age

The project uses Random Forest Regression and a Flask web application for user interaction.
